Nextplate: Ücretsiz Başlangıç Şablonu

Next.js ve TypeScript kullanılarak oluşturulmuştur.

Özellikler

Input
Carousel
Accordion
Button
Toggle
Modal
Kütüphaneler ve Araçlar

Zod: Form validasyon işlemleri için kullanıldı.
Vercel: Deploy işlemleri için kullanıldı.
